Búsqueda de Clientes por Nro. Id en POS

Características:

  • ☑   Este app permite buscar a los clientes por el valor ingresado en el campo "Id. Tributario" desde el Punto de Venta.
  • ☑   Se aplica la misma funcionalidad que aplica al nombre, es decir a medida que se ingresen los números (sin guiones) la lista de clientes se reduce hasta que quede solo aquel que coincida con el número ingresado.
