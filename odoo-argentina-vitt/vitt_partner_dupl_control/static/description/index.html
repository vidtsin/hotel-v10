Control de Números de Documento Duplicados

Funcionalidades:

  • ☑   Este app permite que el usuario especifique el "Tipo de Documento sobre el que debe realizar este control.
  • ☑   El sistema no permitirá guardar un registro de Cliente/Proveedor, si el CUIT/CUIL que se asignó está vinculado a un Tipo de Documento controlado

Configuración

Una vez instalado, es necesario especificar los "Tipos de Documentos" que tendrán aplicado el control.

Aplicación del Control

Si se intenta guardar un registro de Cliente/Proveedor con un número de Documento (CUIT, CUIL, etc.) que ya ha sido ingresado, se mostrará un mensaje de error y no se permitirá guardar el registro.
