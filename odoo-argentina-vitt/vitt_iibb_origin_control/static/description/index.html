Provincia de control para IIBB

Características:

Este app agrega un campo adicional en la Factura de Cliente que registra la Provincia de entrega que se utilizó para el cálculo del impuesto de Ingresos Brutos.

Comportamiento

El campo es completado de forma automática y es editable hasta el momento de validar la factura. El uso del mismo es exclusivamente para realizar controles vinculados a este impuesto.
