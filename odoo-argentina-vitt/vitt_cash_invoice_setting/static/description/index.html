Términos de Pago - Pagos Contado

Características:

Este app permite que al validar la factura con el "Término de Pago especificado, esta quede totalmente pagada sin la necesidad de ingresar un registro de Pagos o Pagos en Grupo.

Configuración

Al instalar el app, un nuevo campo de "Cuenta Contado" es incluido en el registro de "Términos de Pago". Para que al validar la factura, se registre el asiento a una cuenta de Bancos o Caja, es necesario que la cuenta que se ingrese en este campo sea de este tipo.

Al utilizar este "Término de Pago" en una Factura, la Cuenta del notebook "Otra Información" es reemplazada por esta. De esta forma al validar la factura, esta queda pagada de forma automática.


Asiento Contable

El asiento contable de la factura registra la cuenta del "Término de Pago".
