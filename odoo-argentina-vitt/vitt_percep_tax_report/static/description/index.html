Informes Impuestos de Percepciones en Facturas

Observaciones:

  • ☑   Este app instala dos nuevos informes, que muestran los impuestos de percepciones aplicados en facturas de Clientes y/o Proveedores.
  • ☑   Este informe requiere que los impuestos estén correctamente configurados, es decir tengan los "Grupos Impositivos" aplicados.
  • ☑   El formato excel del informe puede incluir vínculos (links) para abrir el registro en Odoo.

Instalación

Al instalar este app, se podrá observar dos nuevos informes en el menú de "Informes ARG" >> "Informes Impositivos", que muestra los dos informes.

Opciones del Informe

  • ☑   Los filtros de Fecha hacen referencia a la fecha de las Facturas.
  • ☑   Es posible filtrar por "Diario de Ventas", y/o "Impuesto de Percepción". Si los filtros no se completan, se incluirá toda la información.
  • ☑   Es posible agrupar por impuesto de percepción, e incluso evitar incluir facturas en las que el impuesto tiene un valor de 0.00.
  • ☑   El formato de impresión de excel, puede incluir un vínculo para abrir el registro en Odoo.

Formatos de Impresión

Este informe puede ser ejecutado para el formato de html, pdf y/o excel.
