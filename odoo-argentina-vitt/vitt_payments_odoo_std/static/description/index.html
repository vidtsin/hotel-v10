Acceso a Pagos y Recibos de Odoo

Funcionalidades:

  • ☑   Este app permite tener acceso a los registros de Pagos estándar de Odoo cuando se ha instalado el app de "Pagos en Grupos".
  • ☑   Se podrán revisar los Pagos y Recibos, pero no será posible generar nuevos registros de forma manual.

Configuración

Una vez instalado, no es necesaria ninguna configuración adicional. Se podrán ver los menús de acceso desde Contabilidad > Compras/Ventas, con los nombres "Recibos Simples" y "Pagos Simples".

Consideraciones

No es posible generar nuevos registros de forma manual.
