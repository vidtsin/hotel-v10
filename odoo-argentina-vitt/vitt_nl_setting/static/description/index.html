Controles Adicionales para Exportaciones Impositivas

Características:

Este app agrega una opción adicional que permite habilitar controles a registros de Facturas de Clientes y/o Proveedores para que las exportaciones impositivas tengan la información completa.

Configuración

Este app funciona en combinación con "Moogah - Configuración, Informes y Exportaciones ARG".
Una vez instalado, muestra la nueva opción "Controles Adicionales ARG"

El contenido de las opciones para control que se muestran puede variar según los apps adicionales que hayan sido instalados.


Controles CITI:

  • ☑   El checkbox "Campos Obligatorios - CITI Compras" agrega controles adicionales para poder validar facturas, como: Filas sin producto no pueden tener Precio ni Impuesto, la Fecha de Vencimiento es de tipo obligatorio, etc.
  • ☑   El checkbox "Campos Obligatorios - CITI Ventas" agrega controles adicionales para poder validar facturas, como: Filas sin producto no pueden tener Precio ni Impuesto, el Término de Pago es obligatorio, la Fecha de Vencimiento es de tipo obligatorio, etc.

Controles Notas de Crédito ARCIBA:

  • ☑   El campo "Impuestos ARCIBA Ventas" permite especificar los impuestos sobre los que se debe realizar el control.
  • ☑   El checkbox "No Calcular Percepciones en NC Parciales y en Meses Distintos" restringe el sistema para que las NC de facturas con Percepciones no realicen el cálculo de este impuesto (siempre que sean de tipo WS) si la fecha de la NC está en un mes distinto que la factura a acreditar y/o es por un valor parcial.
