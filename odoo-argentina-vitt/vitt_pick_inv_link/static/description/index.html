Facturas de Clientes vinculadas a Remitos

Observaciones:

  • ☑   Este app tiene como dependencias: "Secuencia par Remitos Pre-Impresos ARG" (vitt_official_stock_sequence) y VITT ARG Invoice Format (vitt_arg_einvoice_format).
  • ☑   El app requiere que las facturas sean generadas a partir de Ordenes de Ventas con Remitos vinculados.
  • ☑   Se muestra un nuevo notebook en la Factura donde se listan los remitos vinculados.

Instalación

Este app requiere que se tenga configurado al menos un Talonario de Stock para poder asignar los "Nros. Oficiales" en los remitos.

Información en Factura

  • ☑   En el registro de Facturas, se visualiza un nuevo notebook "Pickings".
  • ☑   Es posible tener vinculada m√°s de una Entrega/Remito a la misma factura.

Impresión

Para la correcta impresión de la factura, es decir, incluyendo el detalle del Nro. de Remito, es necesario que el app con el Formato de Impresión de la Factura Electrónica se encuentre instalado.
