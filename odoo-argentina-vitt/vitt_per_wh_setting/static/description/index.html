Campos Impositivos en Provincias

Funcionalidades:

  • ☑   Este app agrega campos adicionales al registro de Provincias relacionados a los impuestos de Percepción y Retención.
  • ☑   Este app debería ser instalado si la empresa necesita registrar este tipo de impuestos.

Configuración y Requerimientos

Este app tiene como dependencias los apps relacionados con el cálculo de impuestos de retención y percepción.

Nuevos Campos

Estos campos son utilizados para realizar los controles correspondientes para el cálculos de los impuestos de Percepción y Retención. Para más detalles sobre la configuración y aplicación, se recomienda revisar la documentación relacionada a estos impuestos.
