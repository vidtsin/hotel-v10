Números Legales en Registros de Stock

Funcionalidades:

  • ☑   Este app permite definir una secuencia especíifica en los Remitos de Stock, permitiendo al usuario editar el número legal en cualquier momento.
  • ☑   El uso está recomendado para los Remitos Pre-Impresos, en cuyo caso y por un tema de control se desea mantener la misma numeración en Odoo

Instalación y Requerimientos

Este app requiere que el módulo de "Stock/Inventarios" se encuentre instalado, adicionalmente, también se sugiere instalar el app que vincula el Número de Remito en la Factura de Clientes.

Configuración

Para que el usuario pueda visualizar el campo, se requiere habilitar el acceso desde el Usuario, seleccionando el campo "Nro. Oficial Remitos".

Configurar Talonario Stock

Esta nueva opción está disponible desde el módulo "Inventario", se deben completar todos los campos de la misma, indicando "Cliente" como 'Tipo de Empresa' para los Remitos, e ingresando el "Número de Líneas" que se debe permitir imprimir en cada Remito.

Asignación en Remitos

Es necesario asignar el "Talonario de Stock", el Número será actualizado en cuanto se ingrese el campo anterior.
