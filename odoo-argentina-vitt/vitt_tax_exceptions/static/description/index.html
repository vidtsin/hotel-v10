Nuevas funcionalidades para Retenciones y Percepciones

Funcionalidades:

  • ☑   Se incorporan el soporte para exenciones de impuestos de retención y percepción.
  • ☑   Soporte para el WS de Moogah para consultar alicuotas de AGIP.
  • ☑   Correcciones al cálculo de Percepciones y Retenciones.
  • ☑   Soporte para controles por Jurisdicción y aplicación de tasas base desde Provincias.

Instalación

Este app tiene como dependencia al módulo de "Manejo de Retenciones Argentina" (l10n_ar_account_withholding). Al instalar este app, se podrá observar un nuevo menú en Contabilidad >> Configuraciones >> Exenciones de Impuestos.

Exenciones de Impuestos

Para configurar una exención para impuestos de Retención y/o Percepción, es necesario seguir los siguientes pasos.

  • ☑   Se debe ingresar la empresa, ya sea un cliente o proveedor.
  • ☑   Se necesita especificar el código de Impuesto de Percepción en el campo "Cód. Impuesto", y el código de Retención en el campo "Cód. Imp. Retención".
  • ☑   Si se configura una Retención de Ganancias, el campo "Código Reg. Ingresos", no obstante el usuario lo podrá editar.
  • ☑   Es necesario indicar si la exención es de tipo Parcial, en cuyo caso se debe indicar el porcentaje para ambos impuestos, o Total.
  • ☑   Como último paso, se debe indicar el período de validez y marcar el registro como "Activo".
